## Common Error: Connection Failed

```
Hi [username],

This error usually means one of the following:
1. The MC Remote API plugin is not installed or activated on the remote site.
2. The API key entered in the connection settings is incorrect.
3. The remote site URL has a typo (check for trailing slash or http vs https).
4. A firewall or security plugin on the remote site is blocking the request.

Steps to troubleshoot:
1. Verify MC Remote API is active on your remote site.
2. Re-generate the API key in MC Remote API → Settings and update it here.
3. Try the "Test Connection" button in MC-Woo Remote Automations → Connections.
4. Check your PHP error log for more details.

Full troubleshooting guide:
https://github.com/Inamo87100/mc-woo-remote-automations/blob/main/mc-woo-remote-automations/docs/TROUBLESHOOTING.md

Let me know what you find!
```
