## Reddit (r/Wordpress, r/woocommerce)

```
Title: I built a free plugin to automate user creation on remote WordPress sites from WooCommerce orders

Hey r/Wordpress!

I just released MC-Woo Remote Automations on WordPress.org.

**The problem it solves**: If you sell access to another WordPress site via WooCommerce,
you previously had to manually create user accounts after each purchase.

**What it does**:
- Hooks into WooCommerce order completion
- Creates a user on a remote WordPress site via REST API
- Assigns the role mapped to the purchased product
- Logs every run with status + timestamps

It requires MC Remote API (also free) installed on the receiving site.

WordPress.org link: https://wordpress.org/plugins/mc-woo-remote-automations/
GitHub (open source): https://github.com/Inamo87100/mc-woo-remote-automations

Happy to answer questions or take feedback!
```
