# Issue Templates Reference

This document summarizes the GitHub issue templates available in this repository.

The actual templates are in [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE/).

---

## Available Templates

| Template | File | Label | Use for |
|----------|------|-------|---------|
| Bug Report | `bug_report.md` | `bug` | Reporting broken functionality |
| Feature Request | `feature_request.md` | `enhancement` | Suggesting new features or improvements |
| Security Vulnerability | `security.md` | `security` | Reporting security issues (non-critical only; email us for critical) |
| Question / Support | `question.md` | `question` | Setup, configuration, and usage questions |

---

## How to Use

1. Go to [**New Issue**](https://github.com/Inamo87100/mc-woo-remote-automations/issues/new/choose).
2. Choose the template that best fits your request.
3. Fill in all required fields.
4. Submit.

---

## Bug Report Template Summary

Required information:
- Description of the bug
- Steps to reproduce
- Expected vs actual behavior
- Environment (WP version, WC version, PHP version, plugin version)
- Error logs if available

---

## Feature Request Template Summary

Required information:
- Problem the feature solves
- Proposed solution
- Alternatives considered
- Use case and acceptance criteria

---

## Security Vulnerability Template Summary

> ⚠️ For **critical** vulnerabilities, email **security@mambacoding.com** directly.

For non-critical issues opened here:
- Brief summary (no full exploit details)
- Affected component and severity
- Affected versions
- Suggested fix (optional)

---

## Question Template Summary

Required information:
- Your question
- What you've already tried
- Environment details

---

## Triage Process

New issues are triaged within **5 business days**:

1. Label applied (`bug`, `enhancement`, `question`, `security`).
2. Priority assigned: `critical`, `high`, `medium`, `low`.
3. Milestone assigned if the fix is scheduled for a specific release.
4. Response provided acknowledging the issue.
